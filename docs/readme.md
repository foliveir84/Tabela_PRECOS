Bem-vindo à documentação oficial do projecto **Verificador de Preços**. Esta pasta contém os guias, padrões e regras arquitecturais do projecto, servindo como referência para qualquer pessoa que desenvolva ou mantenha o código.

3. [Funcionalidades e Módulos](features.md) - Visão geral das integrações (Google Sheets, Sifarma e Infoprex).
4. [Padrões de Exportação](export-standards.md) - Regras para geração de ficheiros Excel e nomenclaturas.
5. [Design System](design-system.md) - Referências visuais e utilização de estilos na UI.
